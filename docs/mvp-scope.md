# MVP Scope Lock

This document locks the MVP scope for the first usable version of the Global Risk Decision Intelligence Platform.

## Countries

- India
- France
- Kenya

## Sectors

- Agriculture
- Energy

## Initial Alerts

- Drought risk
- Crop stress
- Heatwave/grid stress

## Out of Scope For Phase 1

- Data ingestion
- Data source registry
- Risk engine
- Real datasets
- Dashboard analytics
- Maps
- Charts
- AI-generated briefings
- Scenario simulation
- Supply-chain expansion
- Construction expansion
